Code Sample (Python)


def npv(rate, cashflows):
    # cashflows[0] is typically the initial investment (negative)
    return sum(cf / ((1 + rate) ** t) for t, cf in enumerate(cashflows))
print(npv(0.1, [-1000, 400, 400, 400, 400]))

Math: Financial Formulas

Net Present Value (NPV):

$$ \mathrm{NPV} = \sum_{t=0}^{T} \frac{CF_t}{(1 + r)^t} $$

Future Value of an Annuity:

$$ \mathrm{FV} = P \cdot \frac{(1+r)^n - 1}{r} $$

Walk through variables: \(CF_t\) cash flow at time t, r discount rate.
